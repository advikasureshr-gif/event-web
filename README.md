# event-web
Event Notice Board Website 

Created a web-based notice board to display events, and also view them based on category (sports, tech, etc).

# Features
* Displays upcoming and past events
* Countdown timer to show remaining time until an event begins
* Displays appropriate message if event has ended.
* has a demo event registration form for each event to simulate sign-ups.

Technology used: HTML, CSS and JavaScript.

Website was developed as part of academic coursework and was completed as a group project.
Note: Dates used in this project are only for demonstration.
